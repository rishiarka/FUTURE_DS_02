# Customer Churn & Retention Analysis

This project analyzes customer churn patterns to understand why customers leave and how retention strategies can be improved.

## Tools Used
Tableau

## Dataset
Telco Customer Churn Dataset

## Dashboard Insights
- Month-to-month contracts show the highest churn rate.
- Customers with higher monthly charges tend to churn more.
- Most churn occurs within the early tenure period.

## Business Recommendations
- Encourage long-term contracts through discounts.
- Improve onboarding experience for new customers.
- Offer loyalty programs to long-term customers.
